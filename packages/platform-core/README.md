# Platform Core

Platform Core содержит минимальный набор платформенных компонентов,
необходимость существования которых подтверждена независимыми
архитектурными аудитами.

На текущем этапе Platform Core включает:

- interaction-contract

Архитектурные роли Provider и Adapter признаны платформенными,
однако их программные интерфейсы пока не выделяются, поскольку
для этого недостаточно практических подтверждений.

Выделение дополнительных компонентов Platform Core выполняется
только после повторного подтверждения независимыми архитектурными
аудитами.
