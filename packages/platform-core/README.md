## Evolution Policy

Состав Platform Core не является фиксированным.

Новые компоненты включаются в Platform Core только после:

1. независимого архитектурного аудита;
2. подтверждения совпадения архитектурных ролей не менее чем двумя независимыми реализациями;
3. утверждения соответствующего ADR.

Это обеспечивает эволюционное развитие платформы и предотвращает появление преждевременных платформенных абстракций.
